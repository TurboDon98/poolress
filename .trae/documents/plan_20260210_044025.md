# Создание Frontend архитектуры для TurboProject 2.0 (Electron + React)

Я обновил план с учетом ваших требований к полям регистрации. Мы создадим современное desktop-приложение (.exe) с профессиональным дизайном.

## 1. Инициализация проекта и окружения
- [ ] Создать структуру проекта `frontend`.
- [ ] Инициализировать **Vite + React + TypeScript**.
- [ ] Настроить **Electron** для работы в виде десктопного приложения.
- [ ] Настроить **Tailwind CSS** и **Shadcn/UI** для создания интерфейса уровня "IT-гигантов".
- [ ] Установить **Framer Motion** для анимаций.

## 2. Архитектура и Навигация
- [ ] Настроить роутинг и **Sidebar** навигацию (сворачиваемая, с иконками).
- [ ] Реализовать управление состоянием через **Zustand** (хранение данных в памяти сессии).

## 3. Реализация страниц

### Модуль Авторизации (Детально)
- [ ] **Страница Входа:** Email + Пароль.
- [ ] **Страница Регистрации:** Расширенная форма с полями:
    -   Фамилия
    -   Имя
    -   Отчество
    -   Отдел (выпадающий список или ввод)
    -   Почта
    -   Пароль
- [ ] **Страница OTP:** Ввод кода подтверждения (имитация отправки).

### Основные модули
- [ ] **Главная (Dashboard):** Сводная информация и быстрый доступ.
- [ ] **Банк проектов:** Интерактивная таблица проектов (данные на основе вашего маппинга).
- [ ] **Ресурсы:** Управление пулом ресурсов.
- [ ] **Паспорта качества:** Просмотр статусов качества.
- [ ] **Проверка загруженности:** Визуализация графиков (Recharts).
- [ ] **Настройки:** Управление темой и параметрами приложения.

## 4. Сборка и Обновление
- [ ] Настройка **electron-builder** для генерации `.exe`.
- [ ] Подключение модуля **electron-updater** (подготовка клиентской части к автообновлению).

Я настроил конфигурацию под твой репозиторий `TurboDon98/poolress`.

Теперь, чтобы **выпустить обновление**, тебе нужно будет сделать следующие шаги (я не могу сделать их за тебя, так как у меня нет доступа к твоему GitHub аккаунту и токенам):

1.  **Измени версию**: В файле `frontend/package.json` поменяй `"version": "0.1.0"` на новую, например `"0.1.1"`.
2.  **Собери приложение**:
    Открой терминал в папке `frontend` и запусти:
    ```bash
    npm run electron:build
    ```
    Это создаст установщик в папке `frontend/release` (файл типа `TurboProject Setup 0.1.1.exe`).
3.  **Загрузи на GitHub**:
    *   Зайди на https://github.com/TurboDon98/poolress
    *   Справа в разделе "Releases" нажми "Create a new release".
    *   В поле "Tag version" напиши `v0.1.1` (обязательно с 'v').
    *   В заголовок напиши что угодно (например, "Update 0.1.1").
    *   **Самое важное**: Перетащи файл `TurboProject Setup 0.1.1.exe` (и файл `latest.yml`, который тоже появится в папке release) в поле для файлов.
    *   Нажми "Publish release".

После этого, у всех пользователей, у которых установлена версия `0.1.0`, при запуске появится окно: "Доступна новая версия 0.1.1".

**Я всё настроил в коде, тебе останется только собирать и выкладывать файлы на сайт.**
